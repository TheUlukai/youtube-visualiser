# Hegel's Complete Philosophical System

Interactive visualization of Hegel's complete philosophical system — covering
dialectical logic, the Phenomenology of Spirit, the Science of Logic, the
Philosophy of Nature, the Philosophy of Mind, and historical legacy.

**YouTube URL:** https://www.youtube.com/watch?v=XHOMUIkM3nk
**Date generated:** 2026-03-18

## Launch

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Build

```bash
npm run build
```
